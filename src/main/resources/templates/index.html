API Documentation

The API is JSON based, RESTful service that and requires no Authentication Service methods with sample request and response is shown below
Verify Card
This method allows card details to be fetched, this way it can be determined if the card is valid or not
Key Values
URL /card-scheme/verify/{cardbin}
cardbin first 4 or 8 digits of the card
Sample Request 
curl -X GET --header 'Accept: application/json' 'http://localhost:8200/card-scheme/verify/42513433'
Sample Response 
{
  "success": true,
  "payload": {
	"scheme": "visa",
	"type": "debit",
	"bank": "BANK BPH, S.A."
  }
}
Hit Count
This method gives a statistics of number of times each specifc card number was verified
Key Values
URL card-scheme/stats?start={pageNumber}&limit={recordPerPage}
pageNumber Page number of result to return from paginated list
recordPerPage Number of records expected in each page
Sample Request 
curl -X GET --header 'Accept: application/json' 'http://localhost:8200/card-scheme/stats?start=1&limit=2'
Sample Response 
{
  "success": true,
  "payload": {
    "42513433": 3
  },
  "start": 1,
  "limit": 2,
  "size": 1
}
